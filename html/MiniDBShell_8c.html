Detailed Description

File with the implementation of MiniDB Shell.

The implementation of this file is responsible for the shell interpreter of commands to give table manipulation without the need of a query engine. One of the supported commands is the query engine (query your_query).

Since MiniDB can be executed with arguments from command line, command functions support argument count and argument array manipulation. The functions contained in this file are not intended to be called by other code, this are just for the main functionality.
