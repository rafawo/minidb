Detailed Description

File with the functions that permit table manipulation.

This file implements all the funcionts needed to manipulate table creation, modification, and deletion.

All this functions are intended to be used by MiniDB Shell and the query engine of MiniDB (querySemanticAnalyzer[.h/.c])
