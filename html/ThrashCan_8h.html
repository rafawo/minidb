Detailed Description

file that defines the functions that wrap memory allocation.

The purpose of this header is to wrap all the memory allocation done at the query engine, since there are several files that do the work at the same time. With this memory allocation is evaded in a simple way, since all memory is freed calling to tc_emptyThrashcan().
