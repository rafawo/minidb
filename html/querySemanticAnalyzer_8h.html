Detailed Description

File with the front end for the query semantic analyzer used at the syntax analyzer.

Source file with the implementation of semantic analyzer that's used in the syntax analyzer.

RULES:

>You cannot make reference to a column which corresponding table is not present in the query. >You cannot make reference to non existent tables. >Operations in conditional expression should not "operate" condition with different data types.

To make it easier, we'll use our database to do a temporal table named "@temp_table".
